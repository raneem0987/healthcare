# 🏥 Healthcare Analytics Dashboard 

## 📌 Project Overview
This project is an interactive Healthcare Analytics Dashboard built using **Microsoft Power BI**. It provides insights into hospital performance, patient demographics, billing, insurance providers, and medical conditions through dynamic visualizations and KPIs.

The dashboard is designed to help healthcare stakeholders monitor key metrics, identify trends, and make data-driven decisions.

---

## 🎯 Objectives

- Analyze healthcare data efficiently.
- Monitor hospital billing performance.
- Track patient demographics.
- Analyze medical conditions.
- Compare insurance providers.
- Explore admission trends.
- Build an interactive reporting solution.

---

## 📈 KPIs

- 👥 Total Patients
- 💰 Total Billing
- 📊 Average Billing
- 🎂 Average Age
- 💵 Highest Billing
- 💲 Lowest Billing
- 🏥 Average Length of Stay

---

## 📊 Dashboard Features

### Executive Dashboard

- KPI Cards
- Patients by Medical Condition
- Total Billing by Insurance Provider
- Patients by Gender
- Admissions Over Time
- Billing by Medical Condition (Treemap)
- Navigation Button to Detailed Analysis

---

### Detailed Analysis

- Hospital vs Insurance Provider Table
- Medical Condition Slicer
- Gender Slicer
- Admission Type Slicer
- Insurance Provider Slicer
- Interactive Filtering

## 📌 Key Insights

- Billing exceeded **$1.4 Billion**.
- More than **55K patients** were analyzed.
- Average patient age is **52 years**.
- Average length of stay is **16 days**.
- Insurance providers generated comparable billing totals.
- Diabetes and Obesity accounted for the highest billing values.
